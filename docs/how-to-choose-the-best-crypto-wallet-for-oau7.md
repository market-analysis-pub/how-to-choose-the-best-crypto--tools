## Feature Breakdown by Category

### Security Layers Explained
- **Non-Custodial:** You control private keys. Higher risk if lost, maximum control.
- **Custodial:** Exchange/company controls keys. Lower setup friction, higher counterparty risk.
- **Hardware Wallets:** Private keys stored offline. Best for holdings >$5,000 PHP.

### Local Payment Integration (Philippines-Specific)
- **GCash Direct:** Binance, Crypto.com support peer-to-peer GCash deposits.
- **PayMaya Direct:** Crypto.com, selected exchanges offer PayMaya on/off ramps.
- **Workaround:** Most non-custodial wallets require CEX (Binance/Crypto.com) as intermediary.

### BIR Tax Compliance Checklist
- ✅ Wallets to use for compliance: Ledger (export), Trezor (export), Binance (native reports), Crypto.com (tax reports)
- ❌ Poor for BIR audit trail: MetaMask (manual export only), Trust Wallet (limited history)
- **Action:** Export transaction CSV monthly; maintain buy/sell price records in separate spreadsheet.

## Regulatory Status (BIR + BSP as of 2026)
- All listed custodial wallets (Binance, Crypto.com) are monitored under BSP Virtual Asset Service Provider (VASP) framework.
- BIR requires capital gains reporting for all crypto transactions >100K PHP annually.
- Non-custodial wallet usage is legal; exchanges report user transactions.

## Security Audit Checklist Before Selecting a Wallet

- [ ] **2FA enabled** (required for custodial wallets)
- [ ] **Passphrase set** (hardware wallets; Ledger + Trezor support)
- [ ] **Seed phrase backed up** (written, offline, physically secure location)
- [ ] **Withdrawal allowlist** (custodial: whitelist receiving addresses)
- [ ] **Device verification** (non-custodial: mobile app signed, not sideloaded)
- [ ] **Bug bounty program** (wallet provider has active security program: Ledger, Trezor, MetaMask, Binance all qualify)

## Common Mistakes Philippine Users Make

1. **Leaving large balances on exchanges:** Use CEX for trading only; move hodl amounts to hardware or non-custodial wallet.
2. **Mixing custodial and BIR reporting:** Track all wallets in one spreadsheet; custodial wallets auto-report to BIR if verified.
3. **Ignoring GAS fees on withdrawals:** Ethereum withdrawals cost 200-500 PHP; use Layer 2 or USDT Tron for cheaper exits.
4. **No seed phrase backup:** If phone lost and no backup, funds are irrecoverable. Non-negotiable.
5. **Using same password everywhere:** Crypto requires unique, 16+ char passwords per exchange/wallet.
